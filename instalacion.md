Instalación
¿Cómo instalar el ambiente de desarrollo?
Para ejecutar el proyecto en ambiente local necesitas:
Java 17 o superior compatible con tu versión de Spring Boot.
Apache Maven instalado, o usar el Maven Wrapper si tu repositorio ya incluye `mvnw` y `mvnw.cmd`.
Un IDE como IntelliJ IDEA o NetBeans.
Git para clonar el repositorio. Spring Boot 3.2.x requiere Java 17 o superior y soporte explícito para Maven 3.6.3 o posterior. Apache Maven requiere tener un JDK instalado.
