Uso
Sección de referencia para usuario final
Esta sección está orientada al personal del hotel que utilizará el sistema en su operación diaria, por ejemplo recepción o administración.

Acceso al sistema
Abrir un navegador web
Ingresar a la dirección local del sistema: http://localhost:8080/login
Escribir las credenciales de acceso
Presionar el botón Entrar
   
Funcionalidades disponibles:

Consultar huéspedes
Al iniciar sesión, el sistema muestra la pantalla principal con la lista de huéspedes registrados.
En esta vista el usuario puede consultar:
Nombre del huésped
Número de habitación 
Fecha de entrada
Fecha de salida

Registrar un nuevo huésped
Para agregar un nuevo registro:
Hacer clic en Nuevo huésped
Capturar los datos solicitados: 
Nombre
Número de habitación 
Fecha de entrada
Fecha de salida
Presionar Guardar
Si la información es correcta, el sistema regresará al listado principal con el nuevo registro agregado.

Editar un huésped
Para modificar un registro existente:
Ubicar el huésped en la lista
Hacer clic en Editar
Cambiar la información necesaria
Presionar Actualizar 
Eliminar un huésped

Para eliminar un registro:
Ubicar el huésped en la lista
Hacer clic en Eliminar
Cerrar sesión

Para salir del sistema:
Hacer clic en Cerrar sesión
Consideraciones de uso
La fecha de salida no puede ser anterior a la fecha de entrada
Solo usuarios autorizados pueden acceder al sistema
El sistema está diseñado para una administración básica de huéspedes
